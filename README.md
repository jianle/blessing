

* Install [bower](http://bower.io/)

* Environment(Linux)

```
$ sudo apt-get update
$ sudo apt-get install mysql-client mysql-server
$ sudo apt-get install libmysqlclient-dev
$ sudo apt-get install python-dev
```

* Runing

```bash
$ pip install -r requirements.txt
$ bower install
$ python manage.py runserver
```
